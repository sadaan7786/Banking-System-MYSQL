# SmartPayBank — Banking System

MySQL-backed banking web app with Flask REST API and a responsive single-page frontend.

## Tech Stack

Python, Flask, MySQL, HTML/CSS/JS

## Features

* Customer registration with admin approval flow
* Secure login and session management
* Account balance overview across multiple accounts
* Instant fund transfers by mobile number
* Full transaction history with credit/debit filtering
* Beneficiary management for saved contacts
* Admin panel to approve or reject pending customers

## Setup

```bash
pip install Mysql-connector
pip install flask


python app.py
```

Open http://localhost:5000
